Fehler in zu importierendem Eintrag (Ziele)


Beschreibung

Dieses Fenster erscheint, wenn beim Import von Ziellisten mit eddi ein Eintrag gefunden wurde, der unvollständig oder inkorrekt ist. In diesem Fenster kann der Eintrag korrigiert werden.
