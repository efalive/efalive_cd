Standardmannschaften


Beschreibung

Dieses Fenster öffnet sich, wenn Du beim Bearbeiten der Boote auf den Button Standardmannschaft konfigurieren klickst, oder aus dem Menü Administration den Punkt Standardmannschaften auswählst.

Hier kannst Du unter einem Mannschaftsnamen Steuermann und Ruderer einer Mannschaft eingeben, um die Eingabe für häufig identische Mannschaften zu erleichtern.

Wenn hier Mannschaften konfiguriert sind, können diese im efa-Hauptfenster über den Button Mannschaft auswählen ausgewählt werden. Die Mitglieder der Mannschaft werden dann in die Felder Steuermann/Mannschaft eingetragen.

Wenn der Name der Mannschaft einem Bootsnamen entspricht, werden beim Eingeben einer neuen Fahrt für dieses Boot von efa automatisch die Namen dieser Mannschaft in die entsprechenden Felder eingetragen. Für das automatische Eintragen muß die Option Standardmannschaft automatisch eintragen in den Allgemeinen Einstellungen gewählt werden. Ansonsten kann eine konfigurierte Standardmannschaft durch Drücken von <F4> in die Felder eingetragen werden.

Mannschaft
 Der Name der Mannschaft. Falls die Mannschaft an ein bestimmtes Boot gekoppelt ist und (sofern konfiguriert) automatisch bei Auswahl dieses Bootes eingetragen werden soll, muß der Mannschaftsname mit dem Bootsnamen identisch sein. Anderenfalls ist jeder beliebige Name als Mannschaftsname zulässig.
Steuermann
 Name des Steuermanns der Standardmannschaft. Das Feld kann bei ungesteuerten Boote leer bleiben.
Mannschaft 1 ... 24
 Namen der Mannschaft.
Ziel
 Standardziel. Dieses Feld kann auch leer bleiben.
Fahrtart
 Falls es sich bei Fahrten dieser Mannschaft um eine bestimmte Fahrtart (z.B. Training) handelt, kann diese Fahrtart hier angegeben werden. Wird der Eintrag --- keine Auswahl --- gewählt, so wird die Einstellung zur Fahrtart wie sonst auch aus dem vorangegangenen Eintrag übernommen.
Eintrag speichern
 Speichert den Eintrag.
