efa - elektronisches Fahrtenbuch

Was ist efa


Kurzbeschreibung

efa ist ein elektronisches Fahrtenbuch für Ruderer, welches die Eingabe und das Auswerten von Fahrten ermöglicht und vereinfacht. Dazu werden Datenlisten mit allen Mitgliedern, Booten und Fahrtzielen geführt. efa hilft beim Eintragen neuer Fahrten, indem es beispielsweise Eingaben anhand der Datenlisten automatisch vervollständigt und somit mühevolle Tipparbeit erleichtert. Desweiteren bietet efa auch eine Maske zur Eingabe der Fahrten direkt im Bootshaus, so daß die Mitglieder ihre Fahrten direkt in den Computer eingeben können, anstatt ein Papier-Fahrtenbuch zu benutzen.


Auswerten der Fahrten

Zum Auswerten der Fahrten bietet efa zahlreiche Statistikmöglichkeiten: Die Mannschafts- und Bootskilometer lassen sich nach verschiedenen Kriterien auswerten und darstellen. Durch Einschränkungen können nur spezielle Einträge ausgewertet werden. Auf diese Weise läßt sich efa sehr vielseitig und flexibel einsetzen. Auch das Erstellen von Meldungen für die Kilometerwettbewerbe bzw. Abzeichen des DRV und des LRV Berlin erledigt efa. Über das System efaWett lassen sich Meldungen sogar direkt über das Internet einsenden. Die Ausgabe der Daten erfolgt wahlweise im Programm oder als Datei (HTML, PDF, XML, Text, CSV) und kann selbstverständlich auch ausgedruckt werden. Das Layout der Ausgabe kann dabei vom Benutzer völlig frei gestaltet werden.


Meldungen zu Fahrtenwettbewerben über das Internet: efaWett

Der Deutsche Ruderverband und der Landesruderverband Berlin unterstützen das Meldesystem efaWett, mit welchem Teilnehmer für Kilometerwettbewerbe direkt über das Internet gemeldet werden können. Die hierfür benötigten Meldedateien werden von efa automatisch auf Knopfdruck erstellt.
Weitere Informationen zu diesem Thema findest Du auf der efaWett-Homepage.


Einsatz im Bootshaus

Seit Version 1.3.0 verfügt efa über Funktionen, die es auch erlauben, efa direkt im Bootshaus einzusetzen. efa verwaltet dazu Listen aller Boote, die verfügbar, auf Fahrt oder nicht verfügbar sind und ermöglicht es den Mitgliedern, mit wenigen Mausklicks Fahrten direkt in den Computer einzutragen. Auf diese Weise kann das Übertragen von Fahrten aus dem Papier-Fahrtenbuch in den Computer zukünftig entfallen. Diverse Sicherheitsfunktionen sorgen dabei dafür, daß keine Daten verloren gehen oder manipuliert werden können, sowie Fehleingaben möglichst unwahrscheinlich sind. Die auf diese Weise eingegebenen Fahrten lassen sich natürlich ebenfalls wie oben beschrieben auf vielfältige Weise auswerten.


Plattformunabhängigkeit

efa ist komplett in Java geschrieben. Java-Programme sind im Allgemeinen etwas träge; da efa jedoch ein recht kleines Programm ist, läuft es selbst auf betagten Rechnern schnell genug. Der Vorteil an Java liegt in der Plattformunabhängigkeit --- efa ist unter Linux entwickelt, läuft aber auch unter Windows und allen anderen Betriebssystemen, für die eine Java Virtual Machine existiert (s. auch Abschnitt 2).
