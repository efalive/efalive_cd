Browser


Beschreibung

efa verfügt über einen integrierten Browser, der zum Darstellen der Kilometerlisten und sonstigen Statistiken benutzt wird, sowie zum Anzeigen der Dokumentation.

Um eine Statistik im efa-Browser zu betrachten, wähle im Statistik-Fenster die Ausgabeart im Programm (Grafik).


Erklï¿½rung der Buttons und Felder
Blau unterlegter Text in der dargestellten Seite, bei dem der Mauszeiger zu einer Hand wird, stellt eine Verknüpfung zu einer anderen Seite dar. Wird dieser Text angeklickt, so wird die verknüpfte Seite angezeigt.
Springt zurück zur zuvor angezeigten Seite. Dieser Button ist nur aktiv, wenn zuvor eine andere Seite angezeigt wurde.
Spring wieder eine Seite weiter. Dieser Button ist nur aktiv, wenn zuvor eine Seite zurückgesprungen wurde.
Aktuell angezeigte Seite neu laden. Sollte es bei der Darstellung der Seite zu Fehlern kommen, kann ein Klicken dieses Buttons den Darstellungsfehler u.U. beseitigen.
Aktuell angezeigte Seite speichern. Speichert die aktuell angezeigte Seite (sowie eingebettete Bilder im lokalen Verzeichnis) in einer Datei.
Seite ausdrucken. Durch Klicken dieses Buttons wird die aktuell angezeigte Seite ausgedruckt. Hierbei ist zu beachten, daß der Ausdruck immer so auf die Seitenbreite skaliert wird, wie er im Fenster angezeigt wird! Mit anderen Worten: Durch Vergrößern oder Verkleinern des Ausgabefensters wird auch die Skalierung des Ausdrucks beeinflußt. Um den Ausdruck zu Vergrößern, muß das Browserfenster verkleinert werden; wird umgekehrt das Browserfenster verbreitert und es paßt mehr auf die Seite, so wird der Ausdruck kleiner.
Seite einrichten. Durch Drücken dieses Buttons Einstellungen wie Seitenränder u.ä. vorgenommen werden. Die Standardeinstellungen hierzu, die efa verwendet, werden in den Allgemeinen Einstellungen vorgenommen.
Suche
 Suche nach einem Begriff in der angezeigten Seite.
Schließen
 Schließt das Fenster.
