Statistik erstellen


Beschreibung

efa bietet zahlreiche Möglichkeiten, die eingegebenen Fahrten auszuwerten. Es lassen sich verschiedene Arten von Statistiken stellen, sowie die Bedingungen für Kilometerwettbewerbe auswerten. Darüber hinaus kann efa auch Meldedateien für die elektronische Wettbewerbsteilnahme erstellen.

Um in das Fenster Statistik erstellen zu gelangen, wähle aus dem Menü Ausgabe den Punkt Statistiken und Meldedateien erstellen.


Erklï¿½rung der Buttons und Felder
Es gibt drei grundlegend verschiedene Arten von Statistiken: Mannschafts-Kilometer, Boots-Kilometer und Wettbewerbe. Die Einstellungen zu diesen Statistiktypen wird auf den Registerkarten Mannschafts-Kilometer, Boots-Kilometer und Wettbewerbe vorgenommen. Die Registerkarte gespeicherte Statistikeinstellungen dient zum Aufrufen von gespeicherten Statistikeinstellungen.
Im Folgenden werden nun zuerst die Felder besprochen, die immer sichtbar sind, und danach die Felder der einzelnen Registerkarten beschrieben.
Zeitraum für die Berechnung
 Mittels der Felder vom und bis kannst Du die Auswertung auf einen bestimmten Zeitraum eingrenzen. Wenn diese Felder leer bleiben, werden alle Fahrten ausgewertet.
fahrtenbuchübergreifend
 Normlerweise werden nur die Fahrten des aktuellen Fahrtenbuchs ausgewertet. Wenn diese Option gewählt ist, werden auch Einträge aus allen Fahrtenbüchern ausgewertet, die dem aktuellen Fahrtenbuch vorangehen oder das aktuelle Fahrtenbuch fortsetzen (s. Einstellungen zum Fahrtenbuch). Auf diese Weise lassen sich auch über Jahresgrenzen hinweg Fahrten auswerten.
Ausgabeart
 In dieser Auswahlliste kannst Du wählen, in welcher Form die Daten ausgegeben werden. Die verschiedenen Ausgabearten werden im Folgenden kurz erklärt.
  • im Programm (Grafik): Die Daten werden graphisch aufbereitet und direkt im Programm angezeigt. Diese Ausgabeart kann auch zum Ausdrucken der Ergebnisse benutzt werden.
  • im Programm (Text): Diese Ausgabeart zeigt die ausgewerteten Daten in einer schlichten Tabelle direkt im Programm an.
  • im Browser: Bei dieser Ausgabeart werden die ausgewerteten Daten in einem beliebigen externen Webbrowser angezeigt (s. Allgemeine Einstellungen). Auch bei dieser Ausgabeart können die Ergebnisse über die Druckfunktion des Browsers ausgedruckt werden.
  • als HTML-Datei: Bei dieser Ausgabeart erstellt efa eine HTML-Datei, deren Name im Feld Ausgabe in Datei festgelegt wird. Diese HTML-Datei kann ebenfalls zum Ausdrucken benutzt werden, oder beispielsweise in die Vereins-Homepage eingebunden werden. Um nur die Tabelle mit der Kilometerliste in der Datei zu ersetzen und das Dateigerüst beizubehalten, siehe auch Erweiterte Ausgabeeinstellungen.
  • als PDF-Datei: Bei dieser Ausgabeart erstellt efa eine PDF-Datei, die sich mit dem Adobe Acrobat Reader anzeigen läßt. Wenn Du in den Einstellungen zu efa den Pfad zum Acrobat Reader angegeben hast, öffnet efa die erstellte Liste automatisch mit dem Acrobat Reader. Auch diese Ausgabeart kann zum Ausdrucken der Ergebnisse benutzt werden. Für die PDF-Ausgabe benötigt efa das XML-Plugin und das PDF-Plugin, welche efa automatisch aus dem Internet laden kann.
  • als XML-Datei: Diese Ausgabeart erzeugt eine XML-Datei, die beispielsweise für die Weiterverarbeitung der Ausgabedaten durch eine XML-fähige Anwendung benutzt werden kann. Eine DTD für die erzeugte XML-Datei findest Du im ausgabe/layout-Verzeichnis unter dem Namen kilometerliste.dtd. Für die XML-Ausgabe benötigt efa das XML-Plugin, welches efa automatisch aus dem Internet laden kann.
  • als Textdatei: Bei dieser Ausgabeart erstellt efa eine einfache Textdatei.
  • als CSV-Datei: Wenn die ausgewerteten Daten mit einer Tabellenkalkulation weiterverarbeitet werden sollen, kann diese Ausgabeart gewählt werden. efa erstellt dann eine Datei mit Feldern, die durch einen Senkrechtstrich getrennt sind (comma separated values). Diese Datei kann von gängigen Tabellenkalkulationen geöffnet werden.
  • als Meldedatei: Diese Ausgabeart erstellt eine Meldedatei, die zum Melden von Teilnehmern für Kilometerwettbewerbe verwendet werden kann (s. auch Elektronische Wettbewerbsmeldung). Diese Ausgabeart steht nur bei Auswahl der Registerkarte Wettbewerbe zur Verfügung.
Formatierung
 Aus dieser Liste kann eine Formatierung (Layout) für die Ausgabe ausgewählt werden. Neben den mitgelieferten Formatierungen können auch vom Benutzer Layouts für die Ausgabe selbst erstellt werden (s. Erstellen eigener Layouts). Andere Formatierungen als die Standard-Formatierung sind für alle HTML-basierten Ausgabearten möglich (im Programm (Grafik), im Browser, als HTML-Datei), sowie für die PDF- und die XML-Ausgabe. Für andere Ausgabearten steht nur die Standard-Formatierung zur Verfügung.
Ausgabe in Datei
 Wenn im Feld Ausgabeart eine Variante gewählt wurde, bei der efa eine Datei erzeugt, so kann in diesem Feld angegeben werden, in welchem Verzeichnis und unter welchem Namen diese Datei erstellt werden soll. (Beim Erstellen von Meldedateien kann vom Benutzer kein Namen vorgegeben werden.)
Seit Version 1.2.0 kann efa die erstellte Datei auch automatisch auf einen FTP-Server hochladen, um z.B. gleich die Vereins-Homepage zu aktualisieren. In diesem Fall gib in das Feld Ausgabe in Datei den Pfad zur Datei auf dem FTP-Server nach folgendem Schema ein: ftp://nutzername:passwort@mein.server.de/verzeichnis/auf/dem/server/datei.html. Um diese Eingabe nicht jedes Mal vornehmen zu müssen, kann die Statistik natürlich auch über Statistikeinstellungen speichern gespeichert werden. Hierbei ist zu beachten, daß Nutzername und Paßwort im Klartext (d.h. unverschlüsselt) gespeichert werden! Da sie beim FTP-Zugriff aber sowieso unverschlüsselt übertragen werden, ist FTP ohnehin unsicher und das Speichern des Paßworts ist damit dann auch egal. (Für die FTP-Funktionalität benötigt efa ein zusätzliches Plugin, welches beim ersten Benutzen automatisch installiert wird.)
Statistik erstellen
 Durch Betätigen dieses Buttons wird die Statistikerstellung mit den vorgenommenen Einstellungen gestartet.
Mannschafts-Kilometer
 Mannschafts-Kilometer
Bei den Mannschafts-Kilometer werden für alle Ruderer/Innen eines Boots die Kilometer berechnet. Grundlage aller Berechnungen sind also die Ruderer, d.h. pro Fahrt werden (außer im Einer) mehrfach Kilometer berechnet --- wenn die Kilometer im Monat ausgewertet werden, ergeben sich also u.U. andere Ergebnisse, als wenn die Boots-Kilometer zugrunde gelegt werden.
Statistikart
 In diesem Feld wird festgelegt, welche Art von Statistik erstellt werden soll. Es stehen folgende Statistikarten zur Verfügung:
  • Ruderer/Innen: Es werden die Kilometer für jede(n) Ruderer/in berechnet. Die Leistungen eines jeden Ruderers/in werden einzeln aufgeführt.
  • Status: Es werden die Kilometer nach Statusgruppe getrennt berechnet, z.B. Kilometer der Junioren, Senioren usw. Für jede Statusgruppe werden die Leistungen einzeln aufgeführt.
  • Jahrgang: Es werden die Kilometer für die einzelnen Jahrgänge berechnet. Die Leistung jedes Jahrgangs wird einzeln aufgeführt.
  • Geschlecht: Die Kilometer werden getrennt nach männlichen und weiblichen Personen aufgeführt.
  • Ziele: Es werden alle Ziele aufgeführt, zu denen gerudert wurde.
  • Km / Fahrt: Es werden die pro Fahrt geruderten Entfernungen ausgegeben.
  • Monate: Es wird die Ruderleistung pro Monat ausgegeben.
  • Wochentage: Es wird die Ruderleistung pro Wochentag ausgegeben.
  • Tageszeit: Es werden die geruderten Kilometer pro Stunde am Tag ausgegeben.
  • Boote: Es werden für alle (ausgewählten) Ruderer/innen die Bootskilometer berechnet. Dabei geht beispielsweise ein Vierer mit Steuermann fünfmal in die Bewertung ein (im Unterschied zur Statistik Bootskilometer: Boote).
  • Bootsart: Gibt die detaillierte Bootsart-Bezeichnung der geruderten Boote aus.
  • Fahrtart: Gibt die Art der Fahrt aus.
  • Mitruderer/Innen: Es werden die Leistungen aller Personen aufgeführt, die mit einer bestimmten Person, die im Feld nur Name angegeben wird, zusammen gerudert sind.
  • Wer mit Wem: Es wird eine Tabelle erstellt, wer mit wem gerudert ist. In den Tabellenfeldern werden alle ausgewählten Daten (z.B. Kilometer, Fahrten und Km / Fahrt) ausgegeben. Zusätzlich kann hier ausgegeben werden, wer mit wie vielen verschiedenen Personen gerudert ist (Anzahl Verschiedene).
  • Wer Wohin: Es wird eine Tabelle erstellt, in die für jede(n) (ausgewählte(n)) Ruderer/in ermittelt wird, wohin er/sie gerudert ist. Dabei werden in den Tabellenfeldern alle ausgewählten Daten (z.B. Kilometer, Fahrten und Km / Fahrt) ausgegeben. Zusätzlich kann hier ausgegeben werden, wer zu wie vielen verschiedenen Zielen gerudert ist (Anzahl Verschiedene).
  • Wer mit Bootsart: Für jeden Ruderer wird, unterteilt nach der jeweiligen Bootsart, angegeben, wieviel er gerudert ist.
  • Fahrtenbuch: Es wird ein persönliches Fahrtenbuch für einzelne Ruderer/Innen bzw. eine Gruppe von Personen ausgegeben.
  • Jahre: Diese Statistik faßt alle Fahrten eines Jahres zusammen. So lassen sich mehrere Jahre miteinander vergleichen. Um fahrtenbuchübergreifend die Fahrten aus allen Fahrtenbüchern zu berechnen, muß die Option fahrtenbuchübergreifend aktiviert sein.
  • Monatsübersicht: Gibt in einer Tagesdarstellung für jeden Monat die geruderten Kilometer aus.
  • Wer unerlaubt: Bei dieser Statistikart werden alle Ruderer ausgegeben, die in einem Boot gerudert sind, obwohl sie nicht einer der Gruppen angehörten, die dieses Boot rudern dürfen. Es werden hierbei nur Ruderer (keine Steuerleute) berücksichtigt und auch nur dann ausgegeben, wenn die Anzahl der max. Nicht-Gruppenangehörigen für das jeweilige Boot überschritten ist. Die Gruppe, die für das Boot als mind 1 aus dieser Gruppe konfiguriert ist, findet bei dieser Statistikart keine Berücksichtigung.
  • Wer mit Fahrtart: Bei dieser Statistikart wird eine Matrix von Ruderern und ihren Fahrtarten ausgegeben.
Vorjahresvergleich
 Ist diese Option gewählt, werden die Ruderleistungen aus dem gewählten Zeitraum mit denen im selben Zeitraum des Vorjahres verglichen und die Veränderungen (plus/minus) ausgegeben. Wird als Zeitraum 1.1.2003 bis 31.12.2003 angegeben, so bewirkt dies, daß das Jahr 2003 mit dem Jahr 2002 verglichen wird, und zwar im Zeitraum 1.1. bis 31.12. Werden die Felder vom und bis leer gelassen, so wird das aktuelle Jahr mit dem Vorjahr im Zeitraum vom 1.1. bis zum heutigen Tag verglichen.
Hinweis: Da sich die Fahrten des Vorjahres i.d.R. in einem anderen Fahrtenbuch befinden (sollten), muß die Option fahrtenbuchübergreifend ebenfalls gewählt sein!
nur Fahrten berechnen für
 In diesem Bereich des Fensters kann die Auswertung auf bestimmte Personen eingegrenzt werden. Die einzelnen Felder sind im Folgenden beschrieben.
Geschlecht
 Es werden nur Fahrten für die ausgewählten Geschlechter berechnet. Andere Fahrten werden ignoriert.
Status
 Es werden nur Fahrten für Personen berechnet, die einen der ausgewählten Status haben. Personen, die nicht in der Mitgliederliste eingetragen sind, haben immer den Status andere.
Art der Fahrt
 Es werden nur Fahrten berücksichtigt, deren Art in dieser Liste ausgewählt ist.
Auswahl: Name / Gruppe
 Mit dieser Auswahl kann festgelegt werden, ob sich bei dem Eintrag im Feld nur Name/Gruppe um den Namen eines Ruderers oder einer Gruppe handelt.
nur Name / nur Gruppe
 Wenn als Auswahl Name gewählt ist, werden nur Fahrten für die in diesem Feld eingegebene Person berechnet. Auf diese Weise läßt sich beispielsweise zusammen mit der Statistikart Fahrtenbuch ein persönliches Fahrtenbuch für einen Ruderer erstellen. Ist als Auswahl Gruppe gewählt, so beschränkt sich die Auswertung auf alle Mitglieder dieser Gruppe.
Hinweis: Wird als Name $$ eingetragen und die Statistik mit Option Statistik auch im Bootshaus verfügbar machen gespeichert, so kann ein Mitglied beim Aufruf der Statistik im Bootshaus einen konkreten Namen bzw. eine Gruppe eingeben. Wird als Name $? eingetragen, so erfolgt eine Auswertung für das jeweilige Mitglied nur nach Eingabe dessen Paßworts (siehe Mitgliedsdaten).
als Teil eines Namens
 Wenn diese Option aktiviert ist, dann wird der unter nur Name angegebene Name nicht als vollständigen Name betrachtet, sondern nur als Namensbestandteil. So lassen sich z.B. nur Fahrten für die Mitglieder einer Familie auswerten, indem im Feld nur Name der Familienname eingegeben wird.
Ausgabe
 Unter Ausgabe kann festgelegt werden, welche Informationen pro Eintrag ausgegeben werden sollen. Einige spezielle Informationen werden im Folgenden näher erläutert, die anderen dürften selbsterklärend sein.
  • Zielfahrten gibt eine Liste der Zielfahrten aus, wobei einzelne Fahrten durch Schrägstriche getrennt sind, wohingegen mehrere Zielbereiche einer Zielfahrt durch Kommata getrennt sind. Die Ausgabe 3/8/1,2 beispielsweise bedeutet, daß 3 Zielfahrten absolviert wurden, wobei eine Zielfahrt als Zielfahrt für Bereich 3 gewertet werden kann, eine für Bereich 8 und eine für einen der beiden Bereiche 1 oder 2.
  • Wafa-Km gibt die Wanderfahrt-Kilometer (gemäß der Definition des DRV) aus.
  • Km / Fahrt gibt die durchschnittliche, pro Fahrt geruderten Kilometer aus. Sollte eine Mehrtagesfahrt als ein einziger Eintrag und nicht in Form mehrerer Etappen eingegeben worden sein, so ist diese Angabe verzerrt.
  • Dauer (Stunden) gibt die Gesamtzeit auf dem Wasser (Anhand der Abfahrts- und Ankunftszeit jeder Fahrt) an. Sollte eine Mehrtagesfahrt als ein einziger Eintrag und nicht in Form mehrerer Etappen eingegeben worden sein, so ist diese Angabe verzerrt.
  • Km / h gibt die durschnittlichen geruderten Kilometer pro Stunde an. Sollte eine Mehrtagesfahrt als ein einziger Eintrag und nicht in Form mehrerer Etappen eingegeben worden sein, so ist diese Angabe verzerrt.
  • Anzahl Verschiedene (nur bei Statistikart Wer mit Wem und Wer Wohin) gibt an, mit wie vielen verschiedenen Personen (bzw. zu wie vielen verschidenen Zielen) eine Person gerudert ist.
graphische Ausgabe
 Während die Informationen unter Ausgabe nur als reiner Text ausgegeben werden, werden die hier ausgewählten Informationen zusätzliche durch Balkendiagramme veranschaulicht. Der angegebene Wert gibt dabei die Balkenbreite für 100% an. Die Ausgabe von Balken ist nur bei HTML-basierter Ausgabe (im Programm (Grafik), im Browser, als HTML-Datei), sowie für die PDF- und XML-Ausgabe möglich. (Bei den zur Zeit mitgelieferten Layouts für die PDF-Ausgabe wird kein Balken ausgegeben!) Um zu große Balken zu spalten (stauchen), siehe auch Erweiterte Ausgabeeinstellungen.
Sortierung
 Hier wird angegeben, nach welchem Kriterium die Einträge bei der Ausgabe sortiert werden sollen.
Numerierung
 Hier wird angegeben, welche Einträge der Kilometerliste mit einer Nummer in der Spalte Platz versehen werden sollen. Wenn in der Liste Numerierung nichts ausgewählt ist, so wird diese Spalte gänzlich weggelassen.

Boots-Kilometer
 Boots-Kilometer
Bei den Boots-Kilometer werden für alle Boote die Kilometer berechnet. Grundlage aller Berechnungen sind also die Boote, d.h. jede Fahrt wird genau einmal gewertet --- wenn die Kilometer im Monat ausgewertet werden, ergeben sich also u.U. andere Ergebnisse, als wenn die Mannschafts-Kilometer zugrunde gelegt werden.
Statistikart
 In diesem Feld wird festgelegt, welche Art von Statistik erstellt werden soll. Es stehen folgende Statistikarten zur Verfügung:
  • Boote: Es werden die Kilometer für jedes Boot berechnet. Die Benutzung eines jeden Boots wird einzeln aufgeführt.
  • Art: Es werden die Kilometer nach Art der Boote getrennt berechnet und für alle Bootsarten einzeln aufgeführt.
  • Ruderplätze: Es werden die Kilometer nach Anzahl der Ruderplätze berechnet. Die Benutzung der Boote wird nach Ruderplätzen (Einer, Zweier, Vierer usw.) ausgegeben.
  • Ziele: Es werden alle Ziele aufgeführt, zu denen gerudert wurde. Hierbei geht für jedes Boot (unabhängig von der Anzahl der Ruderer/Innen) jedes Ziel genau einmal in die Berechnung ein!
  • Km / Fahrt: Es werden die pro Fahrt geruderten Entfernungen ausgegeben.
  • Monate: Es wird die Bootsnutzung pro Monat ausgegeben. Hierbei geht für jedes Boot (unabhängig von der Anzahl der Ruderer/Innen) jedes Ziel genau einmal in die Berechnung ein!
  • Wochentage: Es wird die Bootsnutzung pro Wochentag ausgegeben. Hierbei geht für jedes Boot (unabhängig von der Anzahl der Ruderer/Innen) jedes Ziel genau einmal in die Berechnung ein!
  • Ruderer: Es werden die Ruderer/Innen ermittelt, die mit einem bestimmten Boot, dessen Name im Feld nur Boot angegeben ist, gerudert sind.
  • Welches Boot Wohin: Es wird eine Tabelle erstellt, in die für jedes Boot ermittelt wird, wohin es gefahren ist. Dabei werden in den Tabellenfeldern alle ausgewählten Daten (z.B. Kilometer, Fahrten und Km / Fahrt) ausgegeben. Zusätzlich kann hier ausgegeben werden, zu wie vielen verschiedenen Zielen das Boot gefahren ist (Anzahl Verschiedene).
  • Fahrtenbuch: Es wird ein Fahrtenbuch für einzelne Boote eine Gruppe von Booten ausgegeben.
  • Jahre: Diese Statistik faßt alle Fahrten eines Jahres zusammen. So lassen sich mehrere Jahre miteinander vergleichen. Um fahrtenbuchübergreifend die Fahrten aus allen Fahrtenbüchern zu berechnen, muß die Option fahrtenbuchübergreifend aktiviert sein.

 Wettbewerbe

 Das oder die Felder unter diesem Punkt legen Kriterien fest, wann ein Mitglied, das die Bedingungen (noch) nicht erfüllt hat, dennoch ausgegeben wird. Diese Angaben betreffen natürlich nicht das Erstellen von Meldedateien, hier werden ausschließlich Mitglieder ausgegeben, die erfüllt haben.
Wettbewerbsbedingungen
 Veranlaßt efa, die Bedingungen für den Wettbewerb bei der Ausgabe mit auszugeben. Dies betrifft nicht das Ertellen von Meldedateien, sondern nur die restlichen Ausgabearten.
nur Zusammenfassung ausgeben
 Diese Option bewirkt, daß nur eine Übersicht über die Mitglieder, die erfüllt haben oder nicht, ausgegeben wird, ohne bei den erfüllenden Mitglieder alle Wanderfahrten aufzuführen.
erweiterte Ausgabe...
 Zeigt die erweiterten Ausgabeeinstellungen an. Dort lassen sich noch einige seltener benötigte Optionen festlegen.
Statistikeinstellungen speichern
 Wenn die aktuellen Einstellungen öfters benötigt werden, so können sie auch gespeichert werden. Sie lassen sich dann über die Registerkarte gespeicherte Statistikeinstellungen jederzeit wieder aufrufen. Um die aktuellen Einstellungen zu speichern, betätige diesen Button.
gespeicherte Statistikeinstellungen
 gespeicherte Statistikeinstellungen
Auf dieser Registerkarte werden alle gespeicherte Statistikeinstellungen angezeigt.
Liste
 In der Liste werden alle gespeicherten Statistikeinstellungen ausgewählt. Sie können mit der Maus einzeln oder gruppenweise durch Klicken mit der Mouse und gleichzeitiges Drücken von <Strg> oder <Shift> ausgewählt werden. Ausgewählte Einstellungen können bearbeitet werden oder zum Erstellen von Statistiken benutzt werden.
Bearbeiten
 Zeigt die Einstellungen einer ausgewählten Statistik an, um sie zu Bearbeiten.
Löschen
 Löscht die ausgewählten Statistikeinstellungen.
