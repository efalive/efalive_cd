elwiz - efa Layout Wizard


Beschreibung

elwiz, der efa Layout Wizard, ist ein Zusatzprogramm von efa, welches dazu dient, Stylesheets (Layouts) für die Ausgabe von efa zu erzeugen. Mit elwiz lassen sich mit nur wenigen Mausklicks neue Stylesheets erzeugen.

Um elwiz zu starten, wähle den entsprechenden Eintrag aus dem Startmenü aus (Windows) oder rufe das Startskript auf (elwiz.bat bzw. elwiz.sh).

elwiz ist in zwei Bereiche unterteilt: In dem einen können alle Einstellungen für HTML-Stylesheets getroffen werden, in dem anderen für PDF.

Stylesheet erstellen
 Erstellt ein Stylesheet. Durch Betätigen dieses Buttons erstellt elwiz ein neues HTML- oder PDF-Stylesheet, je nachdem, welche Registerkarte ausgewählt ist. Damit efa das Stylesheet findet, muß es im Verzeichnis ausgabe/layout/html bzw. ausgabe/layout/pdf gespeichert werden. elwiz schlägt das entsprechende Verzeichnis automatisch vor. Der Dateiname ist gleichzeitig der Name, unter dem efa das Stylesheet als Layout anzeigt.
Hinweis: Ein neu erstelltes Stylesheet taucht in efa erst nach dem erneuten Öffnen des Statistikfensters auf!
