Reservierung


Beschreibung

efa ermöglicht es, Boote für einen bestimmten Zeitraum zu reservieren. Um ein Boot zu reservieren, wähle dieses Boot im Hauptfenster aus, klicke auf Bootsreservierungen und anschließend auf Neue Reservierung.

Administratoren haben auch die Möglichkeit, im Admin-Modus Bootsreservierungen zu bearbeiten.

Einmalige / Wöchentliche Reservierung
 Auswahl, ob das Boot einmalig zu einem vorgegebenem Datum oder wöchentlich, immer an einem bestimmten Wochentag, reserviert werden soll.
Von (Tag)
 Der Tag, ab dem das Boot reserviert sein soll. (Dieses Feld wird nur bei einmaligen Reservierungen angezeigt.)
Wochentag
 Der Wochentag, an dem das Boot reserviert sein soll. (Dieses Feld wird nur bei wöchentlichen Reservierungen angezeigt.)
Von (Zeit)
 Die Uhrzeit, ab der das Boot reserviert sein soll.
Bis (Tag)
 Der Tag, bis zu dem das Boot reserviert sein soll. (Dieses Feld wird nur bei einmaligen Reservierungen angezeigt.)
Bis (Zeit)
 Die Uhrzeit, bis zu der das Boot reserviert sein soll.
Reserviert für
 Der Name der Person oder Gruppe, für den/die das Boot reserviert ist.
Reservierungsgrund
 Der Grund der Reservierung oder der Name der Veranstaltung, für die das Boot reserviert sein soll.
Speichern
 Reservierung speichern.
