Erstellen von Meldedateien für Kilometerwettbewerbe

efaWett im Deutschen Ruderverband

Seit Januar 2005 ist es im Deutschen Ruderverband möglich, Teilnehmer für das DRV-Fahrtenabzeichen sowie die DRV-Wanderruderstatistik elektronisch über das Internet zu melden. Hierbei kommt (wie auch im LRV Berlin) das bereits erprobte Meldesystem efaWett zum Einsatz. Entsprechende Meldedateien, die alle für die Meldung benötigte Daten enthalten, können auf Knopfdruck von efa erstellt werden. Weitere Informationen zur elektronischen Meldung von Teilnehmern für das DRV-Fahrtenabzeichen oder der DRV-Wanderruderstatistik sind im Internet unter http://efa.nmichael.de/efawett.html zu finden..

efaWett im Landesruderverband Berlin

Im Jahr 2002 wurde vom Landesruderverband Berlin ein System zum elektronischen Melden von Teilnehmern für Kilometerwettbewerbe erprobt: efaWett (http://efa.rudern.de/). Dieses System befindet sich nun im Einsatz.

Die nötigen Meldedateien, die über efaWett eingeschickt werden, können mit efa auf Knopfdruck erstellt werden. Wie dies genau funktioniert, beschreibt das Kapitel 5 der Tour. Du kannst die Tour in efa über den Menüpunkt Tour im Menü Info starten, oder Dir das Kapitel direkt hier im Browser anschauen.

Eine weitere Möglichkeit zum Erstellen von Meldedateien, ohne sämtliche Fahrten zuvor in efa eingeben zu müssen, bietet emil, der elektronische Meldedatei Editor. Er ist Bestandteil von efa und wird, ebenso wie efa, über ein Startskript oder unter Windows über einen Eintrag im Startmenü gestartet. In diesem Editor kannst Du die Daten zu den Teilnehmern eingeben, die Du sonst in ein Papierformular eintragen würdest. emil erstellt dann auf Knopfdruck eine entsprechende Meldedatei. Weitere Informationen zur Benutzung von emil erhälst Du, wenn Du emil startest und anschließend im emil-Fenster <F1> drückst.
