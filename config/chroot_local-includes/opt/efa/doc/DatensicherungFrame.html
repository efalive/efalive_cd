Datensicherung


Beschreibung

efa ist in der Lage, alle relevanten, selbst erstellten Dateien in einem ZIP-Archiv zu sichern. Diese Daten können einerseits als Sicherung verwendet werden, andererseits aber auch z.B. auf einen anderen Computer übertragen werden.

Um eine Datensicherung durchzuführen, wähle im Hauptfenster von efa im Menü Datei den Punkt Datensicherung. efa ermittelt daraufhin selbständig, welche Verzeichnisse gesichert werden müssen. Im Normalfall ist diese Auswahl ausreichend, so daß keine weiteren Verzeichnisse mehr hinzugefügt werden müssen.
