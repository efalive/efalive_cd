Beschreibung

efa verwaltet für jedes Fahrtenbuch eine eigene Boots-, Mitglieder- und Zielliste, sowie eine Liste der Mehrtagesfahrten. Desweiteren gibt es eine Liste von Standardmannschaften für jedes Boot sowie eine Liste von Gruppen, in der beliebige Mitglieder zusammengefaßt werden können. Um diese Listen zu bearbeiten, wähle im efa-Hauptfenster aus dem Menü Administration den Punkt Fahrtenbuch und dort einen der Punkte Mitgliederliste, Bootsliste, Zielliste bzw. Mehrtagesfahrten sowie Standardmannschaften oder Gruppen aus dem Menü Administration.

In diesem Fenster werden alle Einträge der aktuell geöffneten Mitglieder-, Boots-, Ziel-, Mehrtagesfahrt-, Standardmannschaften- oder Gruppen-Liste angezeigt. Neue Einträge können hinzugefügt werden, vorhandene Einträge können geändert und gelöscht werden.

Hinzufügen
 Ein neues Mitglied / Boot / Ziel / eine neue Standardmannschaft hinzufügen (bei Mehrtagesfahrten nicht verfügbar).
Bearbeiten
 Die aktuell ausgewählten Einträge bearbeiten.
Löschen
 Die aktuell ausgewählten Einträge löschen.
Liste ausgeben
 Gibt eine Liste mit allen Einträgen aus.
Status aktualisieren
 Aktualisiert den Status aller Mitglieder anhand ihres Jahrgangs und des aktuellen Status.
Schließen
 Schließt das Fenster.
