Neues Boot hinzufügen / Boot bearbeiten


Beschreibung

Dieses Fenster öffnet sich, wenn Du bei der Eingabe der Fahrten auf den Button hinter dem Eingabefeld Boot klickst, oder wenn Du ein Boot unter Administration - Fahrtenbuch - Bootsliste bearbeitest. Hier kannst Du festlegen, um welche Art von Boot es sich handelt.

Um die Eingabe von Booten zu erleichtern, solltest Du alle Boote Deines Vereins zur Bootsliste hinzufügen. Nur so kann efa den Bootsnamen bei der Eingabe automatisch vervollständigen und nur so viele Mannschaftsfelder anspringen, wie tatsächlich Ruderer im Boot Platz haben. Außerdem lassen sich auf diese Weise Statistiken zu den benutzten Bootsarten erstellen.

Bootsdaten
 Bootsdaten
Auf der Registerkarte Bootsdaten werden alle allgemeinen Einstellungen zum Boot vorgenommen.
Bootsname
 Der Name des Boots.
falls auswärtig, Vereinsname
 Name des Vereins, falls es sich um ein fremdes Boot handelt.
Bootsart
 In den vier Listen kannst Du auswählen, um was für eine Art von Boot es sich handelt. Diese Angaben werden zu Statistikzwecken benötigt, sowie für die Cursorsteuerung bei der Eingabe (s.o.).
Kombi-Boot (auch ruderbar als...)
 Diese Auswahlfelder werden nur beim Hinzufügen eines neuen Boots angezeigt, bei dem es sich um ein Kombi-Boot handeln könnte. Wenn Du ein Boot einträgst, das wahlweise als Skull- oder Riemenboot bzw. mit oder ohne Steuermann gerudert werden kann, so aktiviere die entsprechende Option. In diesem Fall werden zwei neue Bootsnamen zur Bootsliste hinzugefügt (für jede Variante ein eigener Name), die beide Synonyme für den eigentlichen Namen sind. Bei der Auswertung wird immer der eigentliche Bootsname ausgegeben.
Standardmannschaft konfigurieren
 Für jedes Boot kann eine Standardmannschaft angegeben werden, die dieses Boot normalerweise rudert (besonders bei festen Trainingsmannschaften). Klicke auf diesen Button, um eine Standardmannschaft anzugeben oder zu verändern.
Rudererlaubnis
 Rudererlaubnis
Auf der Registerkarte Rudererlaubnis kann angegeben werden, von welchen Gruppen dieses Boote genutzt werden darf. Diese Funktionalität ist für den Einsatz im Bootshaus gedacht: efa prüft beim Beginnen einer Fahrt, ob die Bedingungen erfüllt sind und warnt anderenfalls. Außerdem lassen sich Statistiken erstellen von allen Ruderern, die ohne Erlaubnis ein Boot benutzt haben. Wenn sie nicht benötigt wird, können diese Felder leer gelassen werden.
Gruppe n
 In die Felder Gruppe 1 bis Gruppe 5 können bis zu fünf zuvor definierte Gruppen eingetragen werden. Nur Mitglieder dieser Gruppen sind dann berechtigt, dieses Boot zu rudern. Die Bootshaus-Version von efa prüft beim Beginnen einer Fahrt, ob die Ruderer, die im Boot sitzen, auch diesen Gruppen angehören, und warnt anderenfalls.
Hinweis: Die Gruppen beziehen sich nur auf die Ruderer, nicht auf die Steuerleute eines Bootes.
Max. erlaubte Anzahl an Ruderern, die NICHT einer der Gruppen angehören
 In diesem Feld kann festgelegt werden, wieviele der Ruderer im Boot nicht den Gruppen angehören müssen. efa warnt nur, wenn diese Zahl überschritten wird. 0 bedeutet, daß alle Ruderer mindestens einer der genannten Gruppen angehören müssen, 1 bedeutet, daß max. eine Person nicht einer der Gruppen angehören muß usw.
Mindestens ein Ruderer (oder der Steuermann) muß folgender Gruppe angehören
 In diesem Feld kann eine weitere Gruppe angegeben werden, aus der mindestens ein Ruderer oder Steuermann stammen muß. Dies kann verwendet werden um sicherzustellen, daß immer ein Obmann im Boot sitzt (ein Mitglied der Gruppe Obleute).
Weitere
 Weitere
Diese Registerkarte enthält einige weitere Felder, die zur freien Verwendung bestimmt sind. Sie können von Vereinen genutzt werden, die mit efa ihre Bootsverwaltung machen möchten. efa wertet keines dieser Felder aus.
Freie Verwendung
 Ein Feld mit beliebigem Inhalt, das von efa nicht ausgewertet wird.
Boot hinzufügen
 Fügt das Boot zur Bootsliste hinzu. Dieser Button ist nur verfügbar, wenn das Fenster für das Hinzufügen eines neuen Bootes geöffnet wurde.
Eintrag übernehmen
 Speichert die Änderungen an dem Boot. Dieser Button erscheint nur, wenn es sich um ein bereits vorhandenes Boot handelt.
