JAXP-Plugin

Um die XML-Funktionalität von efa nutzen zu können, wird das Java API for XML Processing (kurz JAXP) benötigt. efa verwendet XML intern für alle HTML-Ausgaben, die nicht mit der Standard-Formatierung ausgegeben werden, sowie für alle XML- und PDF-Ausgaben. (Für die PDF-Ausgabe ist zusätzlich FOP nötig.)

JAXP 1.2 kann von http://efa.nmichael.de/plugins/ heruntergeladen werden. Weitere Informationen zu JAXP, sowie die Original-Downloads von Sun, gibt es unter http://java.sun.com/xml/downloads/jaxp.html.

Download und Installation des JAXP-Plugins

  • Lade JAXP in der Version 1.2 oder neuer aus dem Internet, z.B. indem Du folgende Dateien herunterlädst:
    • dom.jar (26.710 Bytes)
    • jaxp-api.jar (27.052 Bytes)
    • sax.jar (26.206 Bytes)
    • xalan.jar (906.806 Bytes)
    • xercesImpl.jar (974.490 Bytes)
  • Kopiere die Dateien dom.jar, jaxp-api.jar, sax.jar, xalan.jar und xercesImpl.jar in das efa/program/plugins-Verzeichnis.
  • Beende efa und starte das Programm anschließend neu. Nun sollte die XML-Funktionalität vorhanden sein.
