JSUNTIMES-Plugin

Um die Zeiten für Sonnenauf- und -untergang berechnen zu können, wird das JSUNTIMES-Plugin benötigt.

Das JSUNTIMES-Plugin kann von http://efa.nmichael.de/plugins/ heruntergeladen werden.

Download und Installation des JSUNTIMES-Plugins

  • Lade das JSUNTIMES-Plugin aus dem Internet, z.B. indem Du folgende Datei herunterlädst:
    • jsuntimes.jar (13.207 Bytes)
  • Kopiere die Datei jsuntimes.jar in das efa/program/plugins-Verzeichnis.
  • Beende efa und starte das Programm anschließend neu. Nun sollte die Funktionalität vorhanden sein.
