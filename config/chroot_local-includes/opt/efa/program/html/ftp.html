FTP-Plugin

Um die FTP-Funktionalität von efa nutzen zu können, wird das FTP-Plugin benötigt. Dieses ermöglicht es, direkt aus efa die erzeugten Dateien auf einen FTP-Server zu laden.

Das FTP-Plugin kann von http://efa.nmichael.de/plugins/ heruntergeladen werden. Weitere Informationen zu dem FTP-Plugin sowie die Original-Downloads von Enterprise Distributed Technologies Ltd, gibt es unter http://www.enterprisedt.com/.

Download und Installation des FTP-Plugins

  • Lade das FTP-Plugin aus dem Internet, z.B. indem Du folgende Datei herunterlädst:
    • ftp.jar (35.388 Bytes)
  • Kopiere die Datei ftp.jar in das efa/program/plugins-Verzeichnis.
  • Beende efa und starte das Programm anschließend neu. Nun sollte die FTP-Funktionalität vorhanden sein.
