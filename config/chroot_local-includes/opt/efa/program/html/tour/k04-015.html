Drucken

Die zweite Möglichkeit, die Ergebnisse zu drucken, besteht darin, die Druckfunktion eines externen Webbrowsers zu benutzen. Wähle dazu als Ausgabeart im Browser und erstelle eine Statistik. Der eingestellte externe Webbrowser wird geöffnet und zeigt die Ergebnisse an. Durch einen Klick auf das Druck-Symbol können die Ergebnisse auch auf diese Weise ausgedruckt werden.
