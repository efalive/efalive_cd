elektronisches DRV-Fahrtenabzeichen [1.5.0]

Seit Version 1.5.0 kann efa Meldedateien für das DRV-Fahrtenabzeichen erstellen, die elektronisch über das Internet eingeschickt werden können. Das Ausfüllen von Meldeformularen von Hand kann damit entfallen. Informationen zum Erstellen und Einsenden von Meldedateien für das DRV-Fahrtenabzeichen gibt es im Internet unter der Adresse http://www.nmichael.de/software/efa/efawett.html.
