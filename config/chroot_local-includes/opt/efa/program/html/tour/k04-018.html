weitere Ausgabearten

Als weitere Ausgabemöglichkeiten bietet efa eine PDF-Ausgabe. Die hiermit erzeugten Dateien können mit dem Acrobat Reader betrachtet werden und eignen sich selbstverständlich auch zum Ausdrucken.

Die Ausgabe als Textdatei erzeugt eine einfache Textdatei.

Die XML-Ausgabe ist dafür gedacht, falls die Daten von einem XML-fähigen Programm weiterverarbeitet werden sollen.

Für die PDF- und die XML-Ausgabe wird ein Plugin benötigt, das efa beim ersten Mal automatisch aus dem Internet laden kann. (Besonders das PDF-Plugin ist recht groß, so daß der Download bei analogen Verbindungen einige Minuten dauern kann.)
