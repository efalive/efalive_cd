Datensicherung [1.7.0]

Schon oft wurde ich gefragt, welche Daten eigentlich gesichert werden müssen, um im Notfall alles wiederherstellen zu können. efa v1.7.0 verfügt jetzt über eine Funktion, die alle relevanten Daten zu einem ZIP-Archiv zusammenpackt und an beliebigem Ort sichert. Neben der Datensicherung läßt sich diese Funktion auch dafür nutzen, um die Daten auf einen anderen Rechner zu transportieren.

Die Datensicherung wird im Menü Datei als Punkt Datensicherung aufgerufen.
