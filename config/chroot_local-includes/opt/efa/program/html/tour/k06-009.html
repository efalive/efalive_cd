Auswertung von 'Steg-Kilometern' [1.2.0]

Seit Version 1.2.0 kann für jedes Ziel zusätzlich eine Option Start und Ziel ist eigenes Bootshaus angegeben werden, die bedeutet, daß Fahrten zu diesem Ziel vom eigenen Bootshaus aus gerudert werden. In den erweiterten Ausgabeeinstellungen kann bei der Statistikerstellung die Auswertung nun auf solche Fahrten eingeschränkt werden, um nur die Steg-Kilometer zu berechnen.

Beim Konvertieren der Zielliste in das neue Format der Version 1.2.0 wurde die Option Start und Ziel ist eigenes Bootshaus automatisch für alle Ziele aktiviert. Vereine, die die Steg-Km auswerten wollen, müssen bei allen Zielen, die nicht vom oder zum eigenen Bootshaus gerudert werden, diese Option wieder deaktivieren. Außerdem müssen wirklich alle Ziele, deren Km als Steg-Km zu werten sind, zur Zielliste hinzugefügt werden (Fahrten zu Zielen, die nicht in der Zielliste stehen, werden ignoriert)!

Vereine, die auf diese Auswertung keinen Wert legen, können die Option Start und Ziel ist eigenes Bootshaus ignorieren.
