efa Online-Update [1.4.0]

efa verfügt jetzt über eine Online-Update-Funktion. Mit dieser Funktion kann efa automatisch prüfen, ob im Internet eine neue Version von efa verfügbar ist und diese auch downloaden und installieren. Eine manuelle Installation von efa ist damit zukünftig nicht mehr nötig (aber natürlich weiterhin möglich). Das efa Online-Update ist über den Menüpunkt efa Online-Update im Menü Datei zu erreichen.
