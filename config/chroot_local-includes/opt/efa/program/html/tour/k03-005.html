Kombiboote

In dem Fenster Neues Boot hinzufügen wähle nun als Bootsart z.B. B-Gig aus. Wähle weiterhin 4er, Skull und ohne Stm.. Bei dieser Auswahl werden Dir zwei Möglichkeiten für Kombi-Boote vorgeschlagen: Skull 3er mit Stm. und Riemen Boot. Wähle die erste Option (auch ruderbar als Skull 3er mit Stm.) aus, indem Du sie anklickst. Es erscheint daraufhin ein Häkchen vor dieser Option.

Füge nun das Boot hinzu, indem Du auf Boot hinzufügen klickst.
