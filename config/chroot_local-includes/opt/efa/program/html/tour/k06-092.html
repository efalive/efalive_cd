efa im Bootshaus [1.8.1]

In der Konfiguration für efa im Bootshaus kann jetzt erzwungen werden, daß beim Eintragen von Fahrten nur solche Boots-, Ruderer- oder Zielnamen von efa akzeptiert werden, die in den entsprechenden Boots-, Mitglieder- und Ziellisten hinterlegt sind.

Das aktuelle Eingabefeld beim Eintragen von Fahrten wird jetzt standardmäßig farblich hinterlegt, damit leichter erkennbar ist, in welchem Feld efa eine Eingabe erwartet. Die farbige Hervorhebung kann in der Konfiguration abgeschaltet werden.
