Formatierung

Für manche Ausgabearten lassen sich neben der Standard-Formatierung noch andere Formatierungen auswählen. Diese Formatierungen können vom Benutzer auch selbst erstellt werden. Weitere Informationen hierüber liefert die Hilfe beim Drücken von <F1> im Statistikfenster.

Wenn nicht die Standard-Formatierung verwendet wird, benötigt efa ein Plugin, um die Formatierung zu erstellen. Dieses Plugin kann efa automatisch aus dem Internet herunterladen.
