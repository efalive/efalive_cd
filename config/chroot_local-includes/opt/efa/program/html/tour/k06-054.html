Bootsschäden [1.6.0]

In der Bootshausversion ist es möglich, beim Eintragen von Fahrten Bootsschäden an den Administrator oder Bootswart zu melden. Ab Version 1.6.0 werden diese Bootsschäden zusammen mit dem Status des Bootes gespeichert. Wenn ein Ruderer eine Fahrt mit einem Boot, für das ein Bootsschaden gespeichert ist, beginnen will, so wird er über den Bootsschaden informiert. Der Administrator kann über die Funktion Bootsstatus bearbeiten den Eintrag zum Bootsschaden eines Bootes wieder löschen.
