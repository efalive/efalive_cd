Tabellenkalkulation

Wenn die ausgewerteten Daten mit einer Tabellenkalkulation weiterverarbeitet werden sollen, kann in efa die Ausgabeart als CSV-Datei gewählt werden. efa erstellt dann eine Datei mit Feldern, die durch einen Senkrechtstrich getrennt sind (comma separated values). Diese Datei kann von gängigen Tabellenkalkulationen geöffnet werden.
