Rudererlaubnis - 1 (efa im Bootshaus) [1.7.0]

Auf Basis der neuen Funktionalität der Gruppen lassen sich nun in efa auch Ruderberechtigungen für einzelne Boote definieren. Es könnten z.B. Gruppen Kinder-Gruppe, Trainingsleute und Masters angelegt werden. Für jedes Boot lassen sich dann bis zu fünf Gruppen angeben, die berechtigt sind, dieses Boot zu rudern. Ein Boot, für das die Gruppe Masters angegeben ist, darf nur von Mitgliedern der Gruppe Masters gerudert werden. Wenn die Gruppen Trainingsleute und Masters angegeben sind, darf das Boot nur von Ruderern gefahren werden, die mindestens einer dieser beiden Gruppen angehören. Ruderer, die keiner der Gruppen angehören, dürfen das Boot nicht rudern. Wenn für ein Boot gar keine Gruppen angegeben sind, darf es von allen gerudert werden.

efa prüft beim Beginnen einer Fahrt, ob alle Ruderer auch den erforderlichen Gruppen angehören. Hierbei läßt sich für jedes Boot festlegen, ab wievielen Ruderern, die keiner der Gruppen angehören, efa warnen soll. Bei der Voreinstellung 0 warnt efa, sobald ein Ruderer keiner der Gruppen angehört. Bei 1 würde efa einen Ruderer, der keiner der Gruppen angehört, tolerieren usw. In jedem Fall wird von efa aber nur eine Warnung ausgegeben -- das Beginnen der Fahrt ist in jedem Fall möglich. Die Prüfung bezieht sich außerdem nur auf die Ruderer und nicht auf die Steuerleute.

Festgelegt werden können die Gruppen in der Bootsliste für jedes einzelne Boot (Admin-Modus - Bootsliste bearbeiten).

In der Statistikauswertung ist eine neue Statistikart Wer unerlaubt hinzugekommen, die alle Ruderer ausgibt, die ein Boot gerudert sind, ohne einer der erforderlichen Gruppen anzugehören. Hierbei werden nur Einträge berücksichtigt, bei denen die maximal zulässige Anzahl von unerlaubten Ruderern überschritten wurde.
