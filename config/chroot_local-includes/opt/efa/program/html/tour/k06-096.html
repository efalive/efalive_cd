Kleinigkeiten [1.8.1]

efa kennt jetzt als neue Bootsart den Typ Inrigger.

Die Auswertung von Statistiken kann auf Einträge von Booten, die nur von einer bestimmten Gruppe gerudert werden dürfen, eingeschränkt werden. So läßt sich beispielsweise die unerlaubte Benutzung von Booten einer bestimmten Kategorie (z.B. Trainingsboote = Boote, die von der Gruppe Trainingsleute gerudert werden dürfen) leichter auswerten.

Darüber hinaus wurden zahlreiche kleine Fehler korrigiert und Unschönheiten beseitigt, sowie einige kleine Erweiterungen und Verbesserungen realisiert.
