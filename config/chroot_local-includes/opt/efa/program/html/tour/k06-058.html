Gruppen [1.7.0]

efa erlaubt jetzt das Festlegen von Gruppen: Eine Gruppe ist die Zusammenfassung von beliebig vielen Mitgliedern unter einem gemeinsamen Namen. Für jede Gruppe läßt sich festlegen, welche Mitglieder ihr angehören. Ein Mitglied kann zu beliebig vielen Gruppen gehören.

Auf Basis von Gruppen ist es möglich, Statistiken zu erstellen oder beim Einsatz im Bootshaus Ruderberechtigungen für Boote zu vergeben. Das Festlegen von Ruderberechtigungen wird ein paar Abschnitte später beschrieben. Hier geht es erstmal nur um das Erstellen von Statistiken.

Über das Feld nur Name konnte schon in früheren Versionen die Erstellung einer Statistik auf eine einzelne Person eingegrenzt werden. Hinzugekommen in v1.7.0 ist ein Schalter, der es erlaubt, statt des Namens eines Mitglieds den Namen einer Gruppe einzugeben. Auf diese Weise läßt sich eine Statistik für alle Mitglieder einer Gruppe erstellen.
