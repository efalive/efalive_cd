Nachrichten per email verschicken (efa im Bootshaus) [1.7.0]

efa erlaubt es, Nachrichten an den Administrator oder den Bootswart zu schicken. Bislang wurden diese Nachrichten nur intern in efa verwaltet. Mit der neuen Version 1.7.0 ist es möglich, diese Nachrichten auch an eine frei konfigurierbare email-Adresse zu verschicken.

Hierzu muß in der efa-Konfiguration auf der Registerkarte efa im Bootshaus - Weitere Einstellungen ein SMTP-Server zum email-Versand konfiguriert werden. In der Verwaltung der Admins kann dann für jeden Admin eine email-Adresse eingetragen werden. Nachrichten, die dieser Admin lesen darf, werden dann zusätzlich auch als email an ihn verschickt.

Hinweis: Für diese Funktionalität ist ein Plugin erforderlich, das efa beim Konfigurieren der Daten automatisch herunterläd.
