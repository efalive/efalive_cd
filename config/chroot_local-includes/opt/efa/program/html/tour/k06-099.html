Sperren von efa [1.8.2]

efa im Bootshaus kann durch den Administrator für einen konfigurierbaren Zeitraum für die Benutzung durch Mitglieder gesperrt werden. Diese Funktion ist gedacht, um zum Beispiel bei einem Ruderverbot wegen Hochwasser oder einer besonderen Veranstaltung vorübergehend den Zugriff von Mitgliedern (oder Gästen) auf efa zu unterbinden.
