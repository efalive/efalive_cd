elektronischer Fahrtenwettbewerb

In diesem Kapitel geht es um das Erstellen von Meldedateien für die elektronische Meldung von Teilnehmern für Kilometerwettbewerbe.

Die elektronische Meldung wird zur Zeit (Stand: Januar 2005) vom Deutschen Ruderverband und vom Landesruderverband Berlin. Aktuelle Informationen vom elektronischen Meldesystem efaWett findest Du im Internet unter http://www.nmichael.de/efaWett/. Dort gibt es auch eine ausführliche Anleitung zu Meldungen für das DRV-Fahrtenabzeichen, die über diese Tour weit hinausgeht.
