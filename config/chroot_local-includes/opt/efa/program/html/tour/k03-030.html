Zugriffsschutz

Falls der PC, auf dem efa installiert ist, im Bootshaus steht, besteht vielleicht der Wunsch, die Daten gegen unberechtigte Veränderung zu schützen. Der Zugriffsschutz kann aber auch als Absicherung gegen eigene, versehentliche Veränderungen an (bspw. bereits abgeschlossenen) Fahrtenbüchern benutzt werden.

Hinweis: Wird die Version efaDirekt für den Einsatz im Bootshaus eingesetzt, so kann das herkömmliche efa komplett gesperrt werden; Änderungen an den Datenlisten sind dann nur noch für Administratoren möglich. In diesem Fall sollte der Zugriffsschutz daher nicht aktiviert werden!
