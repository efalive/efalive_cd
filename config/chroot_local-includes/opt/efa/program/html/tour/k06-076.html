Weitere Neuerungen [1.7.1]

Die Tippfehlerkorrektur wurde verbessert: Sie erkennt jetzt auch, wenn ein Name im falschen Format (z.B. Vorname Nachname statt Nachname, Vorname oder umgekehrt) eingegeben wurde. Außerdem läßt sich die Tippfehlerkorrektur jetzt einzeln für Mitglieder-, Boots- und Zielnamen aktivieren.

Beim Importieren eines efa-Fahrtenbuchs in ein anderes efa-Fahrtenbuch läßt sich der Import jetzt auf solche Fahrten begrenzen, bei denen mindestens ein Ruderer oder Steuermann im Boot einer vorher festgelegten Gruppe angehört.

Bei der Statistikausgabe Fahrtenbuch kann nun in den erweiterten Statistikeinstellungen unterbunden werden, daß die Zielbereiche im Bemerkungs-Feld ausgegeben werden.
