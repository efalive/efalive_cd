FTP Upload von Kilometerlisten [1.2.0]

Vereine, die ihre Kilometerlisten auf der Vereinshomepage veröffentlichen, laden in der Regel nach dem Erstellen der Kilometerliste in efa diese auf den FTP-Server ihres Webspace-Hosters hoch. Dies kann nun auch efa erledigen: Dazu muß lediglich im Statistikfenster in das Feld Ausgabe in Datei statt eines lokalen Dateinamens der Name einer Datei auf dem FTP-Server nach folgendem Schema angegeben werden: ftp://nutzername:passwort@mein.server.de/verzeichnis/auf/dem/server/datei.html
