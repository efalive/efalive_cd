Automatischer Fahrtenbuchwechsel [1.7.3]

In efa sollte für jedes Jahr eine eigene Fahrtenbuchdatei verwendet werden. Das stellt beim Einsatz im Bootshaus den Administrator vor das Problem, zum Jahreswechsel, noch bevor die ersten Neujahrsruderer aufs Wasser gehen, ein neues Fahrtenbuch anzulegen. Mit dieser Version kann efa dies nun automatisch tun.

Um den automatischen Fahrtenbuchwechsel vorzubereiten, gehe in den Admin-Modus, wähle dort Fahrtenbuch auswählen, wähle Für später vorbereiten und trage Datum und gewünschte neue Fahrtenbuchdatei für den Wechsel ein. efa wird dann ohne weitere Eingriffe automatisch zum konfigurierten Datum das Fahrtenbuch wechseln.
