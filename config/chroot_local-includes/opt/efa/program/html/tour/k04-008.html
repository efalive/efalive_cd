Auswertung eingrenzen

Die Auswertung läßt sich auf bestimmte Ruderer/Innen eingrenzen. Alle diesbezüglichen Auswahlmöglichkeiten findest Du links im Fenster unter der Überschrift nur Fahrten berechnen für. Hier kannst Du auswählen, ob beispielsweise nur männliche oder weibliche Mitglieder ausgewertet werden sollen. Außerdem läßt sich die Auswahl auf Ruderer mit einem bestimmten Status eingrenzen. Desweiteren kann gewählt werden, daß nur Fahrten eines bestimmten Typs berücksichtigt werden sollen. Im Feld nur Name kann ein Name angegeben werden, dann werden nur Fahrten für diese Person berücksichtigt. als Teil eines Namens bewirkt, daß z.B. bei der Eingabe eines Nachnamens im Feld nur Name alle Mitglieder mit diesem Nachnamen berücksichtigt werden.

Probiere die Auswirkungen der einzelnen Möglichkeiten nun aus, indem Du bei verschiedener Auswahl Statistiken erstellst.
