Neues Mitglied hinzufügen

Betätige nun den Button Person hinzufügen, um das neue Mitglied in die Mitgliederliste aufzunehmen.

Das Fenster Neues Mitglied hinzufügen schließt sich damit wieder, und der Button hinter dem Namen wird grün. Der Cursor steht automatisch im nächsten Eingabefeld.

Trage nun auf die gleiche Weise weitere Ruderer ein und füge sie zur Mitgliederliste hinzu, bis das Boot "voll" ist. efa springt dann automatisch in das Feld Abfahrt.
