Statistiken [1.7.0]

In v1.7.0 neu hinzugekommen ist die Statistikart Wer mit Bootsart, die für alle Ruderer ihre Kilometer getrennt nach Bootsart in einer großen Matrix auflistet.

Außerdem ist es möglich, sog. anonymisierte Statistiken (z.B. zur Verwendung im Internet) zu erstellen. Hierzu kann anstelle des Namens eine Mitgliedsnummer ausgegeben werden. Um dies zu aktivieren, muß im Statistikfenster über den Punkt erweiterte Ausgabe ... die Option Mitgliedsnummern anstelle von Namen ausgeben aktiviert werden. Außerdem muß für alle Mitglieder in der Mitgliederliste eine Mitgliedsnummer eingetragen werden.

Hinweis: Die Mitgliedsnummer verwendet efa bislang nur zum Zweck dieser Statistik. Eine andere Bedeutung hat das Feld bislang nicht.
