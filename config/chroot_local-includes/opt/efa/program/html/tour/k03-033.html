Fahrtenbuch fortsetzen

In diesem Fall soll kein völlig neues, unabhängiges Fahrtenbuch erstellt werden, sondern das aktuelle Fahrtenbuch fortgesetzt werden. Beim neu Erstellen würde das neue Fahrtenbuch leere Mitglieder-, Boots- und Ziellisten erhalten, während beim Fortsetzen die Listen aus dem aktuellen Fahrtenbuch übernommen werden.

Wähle daher nun den Punkt Fahrtenbuch fortsetzen (Datenlisten übernehmen).
