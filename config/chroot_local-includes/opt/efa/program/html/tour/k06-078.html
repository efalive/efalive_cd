Standardmannschaften [1.7.2]

Bislang waren Standardmannschaften immer an ein Boot gebunden, d.h. für ein Boot konnte jeweils genau eine Standardmannschaft konfiguriert werden, die beim Eintragen einer Fahrt mit dem Boot automatisch oder manuell in die Felder Steuermann/Mannschaft eingetragen wurde.

In der Version 1.7.2 ist diese Kopplung nun aufgehoben: Standardmannschaften können auch unabhängig von Booten über ->Administration->Standardmannschaften eingerichtet und unter beliebigen Namen gespeichert werden. Wenn Standardmannschaften konfiguriert sind, kann über einen neuen Button Mannschaft auswählen im efa-Hauptfenster die gewünschte Mannschaft ausgewählt werden. Ihre Mitglieder werden dann in die Felder Steuermann/Mannschaft eingetragen. Auf diese Weise lassen sich belebig viele Standardmannschaften konfigurieren, die flexibel in beliebigen Booten eingetragen werden können und so die Eingabe von Ruderernamen bei häufig gleichen Mannschaften erleichtern können.
