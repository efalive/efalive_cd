Wettbewerbsausgabe innerhalb von Kilometerlisten [1.2.0]

Bisher konnten Wettbewerbe nur als alleinstehende Statistiken ausgewertet werden. Seit Version 1.2.0 ist es möglich, auch innerhalb einer normalen Kilometerliste in einer zusätzlichen Spalte für jedes Mitglied seinen persönlichen Stand bzgl. eines Wettbewerbs auszugeben. Die nötigen Einstellungen hierzu findest Du im Statistik-Fenster über den Button erweiterte Ausgabe... auf der Registerkarte Wettbewerbe.
