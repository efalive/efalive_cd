Strengeres Verhalten im Bootshaus [1.7.2]

Mit Version 1.7.2 bringt efa einige Konfigurationsparameter mit, die ein etwas strengeres Verhalten beim Einsatz im Bootshaus erzwingen. In der Konfiguration läßt sich jetzt festlegen, daß efa die Auswahl eines Obmanns bei allen Fahrten erzwingt, das Ändern der vorgegebenen Uhrzeiten verhindert, das Ändern der Kilometer für bekannte Ziele verhindert und/oder verhindert, daß mehr Ruderer für ein Boot eingetragen werden, als das Boot Plätze hat.
