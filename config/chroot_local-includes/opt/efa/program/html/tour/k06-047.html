Button für beliebige Kommandos [1.5.2]

Über die Allgemeinen Einstellungen von efa läßt sich in der Bootshaus-Version im Hauptfenster ein weiterer Button einblenden, der mit beliebigem Text versehen werden kann und ein beliebiges Betriebssystemkommando ausführen kann. Diese Funktion kann genutzt werden, um den Mitgliedern im Bootshaus das Starten eines externen Programms zu ermöglichen.
