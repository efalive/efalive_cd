Elektronische Wettbewerbsmeldung [1.8.1]

Die elektronische Wettbewerbsmeldung wurde in mehrerer Hinsicht verbessert. Insbesondere zeigt efa jetzt eine ausführliche Anleitung mit allen notwendigen Schritten zum Einsenden der Ergebnisse an und bietet die Möglichkeit, eine erstellte Meldedatei zum späteren Einsenden abzuspeichern. Letzteres ist insbesondere dann hilfreich, wenn der PC, auf dem efa läuft, keinen Internet-Anschluß hat: In diesem Fall kann die erstellte Meldedatei leicht auf einem USB-Stick abgespeichert werden und auf einen anderen Computer mitgenommen werden.
